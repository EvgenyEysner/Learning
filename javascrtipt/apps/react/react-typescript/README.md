# 🚀 React TypeScript Todo App

[![TypeScript](https://img.shields.io/badge/TypeScript-4.9.5-blue?logo=typescript)](https://www.typescriptlang.org/)
[![React](https://img.shields.io/badge/React-18.2.0-61dafb?logo=react)](https://reactjs.org/)
[![React Router](https://img.shields.io/badge/React_Router-6.15.0-CA4245?logo=react-router)](https://reactrouter.com/)
[![Materialize CSS](https://img.shields.io/badge/Materialize_CSS-1.0.0-eb7077)](https://materializecss.com/)

A simple, yet powerful Todo application built with React and TypeScript. Perfect for learning TypeScript with React! 💡

## ✨ Features

- ➕ Create new todos
- ✅ Mark todos as completed
- 🗑️ Delete todos
- 💾 Persistent storage using localStorage
- 🎨 Clean UI with Materialize CSS
- 📱 Fully responsive design
- 🛣️ React Router navigation

## 🛠️ Technologies

- React 18
- TypeScript 4
- React Router 6
- Materialize CSS
- LocalStorage API

## 🚀 Quick Start

```bash
# Clone this repository
git clone https://github.com/EvgenyEysner/react-typescript.git

# Install dependencies
npm install

# Start development server
npm start
```
